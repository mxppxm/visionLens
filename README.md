# Vision Lens 📸✨

**智能百科知识助手 - 一拍即知，读书无忧！**

专为阅读爱好者打造的 AI 拍照百科应用，让你的摄像头变成随身的智能百科全书！

## 🚀 项目简介

Vision Lens 是一个基于 React 的智能拍照百科 Web 应用，专注于：

- **📖 拍照百科** - 通过摄像头拍摄任何内容，瞬间获得详细的百科知识解答
- **🤖 多 AI 模型支持** - 集成 Google Gemini 和字节跳动豆包模型，知识解答更准确
- **⚡ 快速响应** - 支持并发分析，3-5 秒内获得详细百科信息
- **📚 知识记录** - 自动保存所有百科问答，方便回顾学习
- **🎯 广泛适用** - 支持文字、图片、物品等各种内容的百科询问

**核心用法：对准感兴趣的内容 → 点击拍摄 → 立即获得百科知识！**

**特别适合：微信读书中的百科问题、日常生活中的知识疑问、学习中的概念理解**

## ✨ 功能特性

### 📖 拍照百科核心功能

- **🌍 万物百科识别** - 支持文字、图像、物品、概念等各种内容的百科询问
- **📷 高清摄像头预览** - 支持后置摄像头，适合拍摄书本、屏幕、实物等
- **🚀 一键获取百科** - 点击按钮或按空格键即可拍摄，3-5 秒获得详细百科知识
- **🧠 多 AI 引擎并发** - 同时调用多个 AI 模型，确保知识解答的准确性和全面性
- **⚡ 超快响应速度** - 图片自动压缩优化，AI 响应时间实时显示

### 🤖 AI 模型支持

- **Google Gemini 2.5 Flash** - Google 最新视觉理解模型，百科知识丰富全面
- **字节跳动豆包模型** - 国产大模型，中文百科内容理解能力更强
- **智能模型切换** - 支持多模型配置，不同内容类型选择最佳 AI 引擎
- **并发分析模式** - 可同时使用多个模型分析，综合给出最完整的百科解答

### 📚 知识管理功能

- **📖 详细百科解答** - 不仅回答问题，更提供完整的背景知识和相关信息
- **📝 历史记录管理** - 自动保存所有百科问答，支持分页浏览和搜索
- **🎨 高亮关键内容** - 智能识别并高亮显示重要的百科知识点
- **⏱️ 性能监控** - 实时显示图片处理时间和 AI 响应耗时

### 📱 微信读书特别优化

- **📖 阅读伴侣** - 完美适配微信读书的百科问题场景
- **🔍 快速查询** - 遇到不懂的概念、人物、事件时，一拍即知
- **📚 知识拓展** - 不仅回答当前问题，还提供相关的知识扩展

### 🔧 技术特性

- **📱 移动端优化** - 渐进式 Web 应用，可安装到手机桌面使用
- **🗄️ 本地数据存储** - 使用 IndexedDB 安全存储，保护用户隐私
- **🎨 响应式设计** - 完美适配手机、平板、电脑等各种屏幕
- **🛠️ 智能错误处理** - 网络异常、API 错误等情况的友好提示和自动恢复

## 🛠️ 技术栈

- **前端框架**: React 18 + Hooks
- **构建工具**: Vite (超快热重载)
- **样式框架**: Tailwind CSS
- **数据库**: IndexedDB (通过 idb 库)
- **AI 服务**: Google Gemini API + 字节跳动豆包 API
- **摄像头**: WebRTC getUserMedia API
- **图像处理**: Canvas API (自动压缩优化)
- **并发处理**: Promise.all (多模型同时分析)

## 📸 如何拍照获取百科知识 (核心用法)

### 🚀 三步开启智能百科之旅

#### 第一步：发现感兴趣的内容 📋

- **微信读书场景** - 阅读时遇到不熟悉的概念、人物、地名、专业术语
- **日常生活场景** - 看到不认识的植物、动物、建筑、艺术品等
- **学习场景** - 遇到需要了解背景知识的概念、事件、人物

#### 第二步：对准拍摄 📷

- 打开应用，允许摄像头权限
- 将感兴趣的内容完整地放入摄像头预览框中
- 确保文字或物品清晰、无反光、无遮挡
- **拍摄方式**：
  - 🖱️ 点击页面中央的蓝色拍摄按钮
  - ⌨️ 直接按键盘空格键快速拍摄

#### 第三步：获得百科知识 ✨

- 拍摄后等待 3-5 秒，AI 正在分析内容
- 自动显示详细的百科解答和相关知识
- 所有百科记录自动保存，方便后续回顾学习

### 💡 最佳拍摄技巧

#### 📝 拍摄环境

- **光线充足** - 避免阴影和强烈反光，确保内容清晰可见
- **背景干净** - 减少干扰，突出要查询的内容
- **稳定拍摄** - 避免抖动模糊，确保文字和图像清晰

#### 🎯 内容摆放

- **完整显示** - 确保要查询的内容完全在画面内
- **正向摆放** - 避免倾斜和颠倒，方便 AI 识别
- **聚焦重点** - 一次拍摄一个概念或物体，效果更好

#### 📱 设备建议

- **推荐使用手机** - 后置摄像头拍摄效果最佳
- **适当距离** - 距离内容 15-25cm，确保清晰度
- **多角度尝试** - 如遇识别困难，可尝试不同角度拍摄

## 📦 安装和部署

### 环境要求

- Node.js >= 16.0.0
- 现代浏览器（支持 getUserMedia API）
- HTTPS 环境（摄像头权限要求）

### 快速开始

1. **克隆项目**

```bash
git clone https://github.com/your-username/visionLens.git
cd visionLens
```

2. **安装依赖**

```bash
yarn install
# 或
npm install
```

3. **配置 API Key**

   - 访问 [Google AI Studio](https://aistudio.google.com/)
   - 创建新的 API Key
   - 在应用界面中点击"设置 API Key"按钮
   - 输入你的 API Key

4. **启动开发服务器**

```bash
yarn dev
# 或
npm run dev
```

5. **访问应用**
   - 打开浏览器访问 `http://localhost:5173`
   - 允许摄像头权限
   - 开始拍照和提问！

### 生产部署

```bash
# 构建生产版本
yarn build
# 或
npm run build

# 预览生产版本
yarn preview
# 或
npm run preview
```

## 🔑 AI 模型配置

### 支持的 AI 模型

本应用支持多种 AI 模型，你可以根据需要选择最适合的解题引擎：

#### 🤖 Google Gemini 模型

- **模型**: Gemini 2.5 Flash
- **特点**: 响应速度快，多语言支持优秀
- **适用**: 英语题目、理科计算题

**获取步骤**：

1. 访问 [Google AI Studio](https://aistudio.google.com/)
2. 登录 Google 账户并创建项目
3. 生成 API Key 并复制保存

#### 🇨🇳 字节跳动豆包模型

- **模型**: doubao-seed-1-6-250615 系列
- **特点**: 中文理解能力强，解题思路详细
- **适用**: 中文题目、语文阅读、文科题目

**获取步骤**：

1. 访问 [火山引擎控制台](https://console.volcengine.com/)
2. 开通豆包大模型服务
3. 创建 API Key 并复制保存

### 📱 在应用中配置

1. **打开设置** - 点击应用右上角的"设置"按钮
2. **选择模型** - 从下拉菜单选择想要使用的 AI 模型
3. **输入 API Key** - 粘贴对应模型的 API Key
4. **保存配置** - API Key 将安全保存在本地浏览器中

### 🔧 高级配置

- **并发分析** - 可设置同时使用多个模型分析同一题目
- **智能切换** - 根据题目类型自动推荐最适合的模型
- **模型对比** - 在历史记录中可以看到不同模型的解答差异

> ⚠️ **隐私保护**: 所有 API Key 仅保存在你的本地浏览器中，不会上传到任何服务器。请妥善保管你的 API Key，避免泄露。

## 📱 百科查询技巧与最佳实践

### 🎯 不同内容类型的拍摄技巧

#### 文字内容

- **概念术语**: 确保关键词汇清晰可见，包含适当上下文
- **人物姓名**: 重点拍摄人名，如有简介可一并拍摄
- **地名**: 完整拍摄地名，如有相关描述可适当包含
- **典故引用**: 拍摄完整的典故内容，便于理解语境

#### 图像内容

- **艺术作品**: 确保作品主体完整，如有标题或说明一并拍摄
- **历史文物**: 突出文物特征，包含任何可见的文字说明
- **自然景观**: 重点关注特征景观，避免过多无关背景
- **建筑物**: 拍摄建筑特色部分，如有铭牌或说明牌可一并拍摄

#### 实物识别

- **动植物**: 重点拍摄特征部分，确保特征清晰可辨
- **器物用品**: 突出物品特色和关键细节
- **食物料理**: 拍摄食物外观，如有菜名标识更好

### ⚡ 快速百科查询流程

1. **📋 内容定位** - 确定要查询的内容，移除干扰物
2. **📸 快速拍摄** - 空格键一键拍摄，无需等待
3. **⏱️ 耐心等待** - AI 分析需要 3-5 秒，复杂内容可能稍长
4. **📖 查看解答** - 获得详细百科知识，必要时可切换 AI 模型对比
5. **💾 自动保存** - 所有百科记录自动保存，随时回顾

### 🔍 百科解答质量优化

- **内容完整性** - 确保要查询的内容信息完整，避免截取不全
- **多模型对比** - 对于复杂概念，建议使用多个 AI 模型分析
- **关键词突出** - 将重要的名词、概念用手指指出后拍摄
- **上下文包含** - 适当包含前后文，帮助 AI 理解语境和含义
